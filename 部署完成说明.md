## 🚀 使用方法

### 方法1：使用启动脚本（推荐，解决Windows编码问题）

```powershell
cd D:\horizon\Horizon
python run_horizon.py
```

### 方法2：使用uv run

```powershell
cd D:\horizon\Horizon
uv run horizon
```

### 方法3：指定时间窗口

```powershell
# 抓取最近1小时的内容
python run_horizon.py --hours 1

# 抓取最近48小时的内容
python run_horizon.py --hours 48
```

## ⚙️ 配置说明

### 修改AI服务

如果需要切换到其他AI服务，编辑 `data/config.json`：

#### 切换到DeepSeek：
```json
"ai": {
  "provider": "openai",
  "model": "deepseek-chat",
  "base_url": "https://api.deepseek.com",
  "api_key_env": "OPENAI_API_KEY",
  "temperature": 0.3,
  "max_tokens": 8192
}
```

#### 切换到OpenAI官方：
```json
"ai": {
  "provider": "openai",
  "model": "gpt-4",
  "api_key_env": "OPENAI_API_KEY",
  "temperature": 0.3,
  "max_tokens": 4096
}
```

### 修改信息源

编辑 `data/config.json` 中的 `sources` 部分：

#### 添加GitHub用户：
```json
"github": [
  {
    "type": "user_events",
    "username": "你感兴趣的GitHub用户名",
    "enabled": true
  }
]
```

#### 添加RSS订阅：
```json
"rss": [
  {
    "name": "博客名称",
    "url": "RSS链接地址",
    "enabled": true,
    "category": "分类"
  }
]
```

### 调整过滤设置

```json
"filtering": {
  "ai_score_threshold": 7.0,    // 提高阈值，只保留高质量内容
  "time_window_hours": 48       // 扩大时间窗口
}
```

## ⚠️ 常见问题

### 1. Windows编码问题
如果遇到 `UnicodeEncodeError`，使用启动脚本：
```powershell
python run_horizon.py
```

### 2. API密钥错误
检查 `.env` 文件中的密钥是否正确配置。

### 3. 依赖包缺失
重新安装依赖：
```powershell
uv sync
```

### 4. 网络连接问题
确保网络可以访问：
- 星火API：https://spark-api-open.xf-yun.com
- GitHub API：https://api.github.com
- 其他信息源
