## 🚀 如何使用项目

### 本地运行

#### 方法1：使用 uv run（推荐）
```powershell
cd D:\horizon\Horizon
uv run horizon --hours 1    # 测试配置
uv run horizon              # 生成完整日报
```

#### 方法2：使用启动脚本
```powershell
cd D:\horizon\Horizon
python run_horizon.py --hours 1
```

### 自动化运行（GitHub Actions）

#### 快速配置步骤：

1. **运行自动化脚本**
```powershell
cd D:\horizon\Horizon
.\setup_github.ps1
```

2. **在 GitHub 上配置 Secrets**
   - 访问你的 GitHub 仓库
   - Settings → Secrets and variables → Actions
   - 添加 `SPARK_API_PASSWORD` 和 `GITHUB_TOKEN`

3. **启用 GitHub Actions**
   - 点击 Actions 标签
   - 启用工作流

4. **手动触发测试**
   - Actions → Daily Horizon Summary → Run workflow

#### 详细配置指南：
查看 `GitHub自动化配置指南.md` 文件

## 💡 常见问题解决

### 问题1：Windows 编码错误
**解决方案**：使用启动脚本
```powershell
python run_horizon.py --hours 1
```

### 问题2：依赖包缺失
**解决方案**：重新安装依赖
```powershell
uv sync
```

### 问题3：API 密钥错误
**解决方案**：检查 `.env` 文件配置
```powershell
python test_config.py
```

### 问题4：GitHub 推送失败
**解决方案**：使用 GitHub Token 认证
```powershell
git remote set-url origin https://你的Token@github.com/用户名/仓库名.git
git push -u origin main
```
